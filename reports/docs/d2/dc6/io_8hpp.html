Detailed Description

Contains IO stream manipulator \forcedlink{io} for types. If type is streamable using its own operator or its conversion operator, then the types operator is used.

Example:

#include <pfr/io.hpp>
struct comparable_struct { // No operators defined for that structure
int i; short s; char data[7]; bool bl; int a,b,c,d,e,f;
};
// ...
comparable_struct s1 {0, 1, "Hello", false, 6,7,8,9,10,11};
std::cout << pfr::io(s1); // Outputs: {0, 1, H, e, l, l, o, , , 0, 6, 7, 8, 9, 10, 11}
